CODING PATTERNS -- You MUST follow these:
- Components: React.forwardRef + React.memo, typed props interface
- Screens: export default function in packages/app/{feature}/screen.tsx
- Pages: thin wrappers in apps/*/pages/ that import screens from packages/app
- Hooks: custom hooks in packages/app/apiHooks/ using TanStack Query
- Styles: StyleSheet.create() at bottom of file, theme via useTheme() hook
- API routes: itty-router handlers in api/src/services/{domain}/{domain}.routes.ts
- Validation: Zod schemas in api/src/services/{domain}/{domain}.schema.ts
- DB queries: Kysely query builder (type-safe SQL), NOT raw SQL strings
- Cross-platform: Use React Native components (View, Text, Pressable), NOT HTML divs/spans
- Imports: Use @shop-diary/ui for UI components, @shop-diary/app for shared logic

EXAMPLE -- API route:
```ts
// api/src/services/shop/shop.routes.ts
import { Router } from 'itty-router'
import { withAuthenticatedUser } from '../../infra/auth'
import { getDb } from '../../infra/db'

const router = Router({ base: '/shop' })

router.get('/', withAuthenticatedUser, async (req, env) => {
  const db = getDb(env)
  const shop = await db.selectFrom('shops').where('id', '=', req.user.shopId).selectAll().executeTakeFirst()
  return Response.json(shop)
})

export default router
```
